# Claim Prediction Project

**Predicting insurance claims using machine learning models**

---

## Project Overview
This project focuses on predicting whether a customer will make an insurance claim. Using structured data, the project explores patterns between customer features and claim outcomes to build a predictive machine learning model.

---

## Dataset
- `Train_data.csv` — dataset used for training the model  
- `Variable Description.csv` — details of each column/feature  

---

## What I Did
- Cleaned and preprocessed data using **Python** and **Pandas**  
- Explored data and visualized patterns to understand relationships  
- Built machine learning models for **classification**  
- Evaluated models using **accuracy, confusion matrix, and other metrics**  
- Iterated on features and model parameters to improve performance
